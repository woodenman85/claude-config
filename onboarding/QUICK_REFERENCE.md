# Quick Reference — Claude for Insurance Agents

Your daily cheat sheet. Copy any of these prompts word-for-word.

---

## 🌅 Start Every Morning

```
Good morning. Pull my daily briefing — GHL pipeline, today's calendar, and any lead emails that need attention.
```

---

## 📞 New Lead Just Came In

```
I just got a new [life insurance / mortgage protection / final expense] lead named [Name]. Help me reach out right now — draft a first text and a voicemail script.
```

```
Find [Name] in my GHL and show me their lead info and any notes.
```

---

## 🔁 Lead Gone Cold

```
[Name] filled out my life insurance form [X] days ago and hasn't responded. What should I send them next?
```

```
Draft a re-engagement text for a final expense lead who went quiet 2 weeks ago.
```

---

## 📅 Appointments

```
Set a reminder in my calendar for a call with [Name] tomorrow at [time]. Label it "[Name] — Life Insurance Consult" and set a 24hr reminder.
```

```
Send [Name] a reminder text — we have a call tomorrow at [time].
```

## 📧 Emails

Search my inbox for any unanswered emails from leads in the last 3 days.

## 🤝 Referrals

```
Draft a referral ask text to send to [Name] — they just got their policy set up.
```

```
Write a referral request email — something warm and non-pushy for a happy life insurance client.
```

---

## 💡 Tips

- **Be specific** — names, dates, product types help Claude give better answers
- **Paste content** when asking for compliance checks or rewrites
- **Chain requests** — "draft a text AND a voicemail for the same lead" works great
- **Ask for options** — "give me 3 versions of this subject line"
