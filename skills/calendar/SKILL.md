# Calendar Skill

## Tools available
Use `mcp__*__*` Calendar tools directly.

## Insurance agency use cases

### Booking appointments
- "Schedule a consultation call with [name] for Thursday at 2pm"
- "Block 30 minutes tomorrow morning for prospecting calls"
- "Add a follow-up reminder for [lead name] in 3 days"

### Checking schedule
- "What does my week look like?"
- "Do I have anything booked Friday afternoon?"
- "Show me all my client calls this week"

### Planning
- "Block every Monday 9-11am for lead follow-up calls"
- "Schedule weekly GHL audit time on Fridays"

## Best practices
- Always confirm the time and date before creating events
- Add client name and product type to event title (e.g. "John Smith — Mortgage Protection Consult")
- Set reminders 24hr and 1hr before client appointments
- Never delete events without confirming with the user
